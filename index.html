Features

Built for real dashboard workflows.

Designed for admin panels, CRM tables, finance reports, inventory systems, and internal tools that need fast annotation without backend storage.

📋

Copas Mode

Select table cells or page content and copy it quickly with keyboard shortcuts.

🏷️

Labels Mode

Add temporary colored labels to cells, cards, buttons, text, and other elements.

📌

Pin Badges

Labeled elements show small colored badges, making review work easier to track.

🔎

Label Filter Bar

Filter labeled elements by status, priority, review type, or your own custom labels.

⚡

Keyboard First

Press Ctrl/Alt + T to open tools, Ctrl + C to copy, and Ctrl + Shift + X to clear labels.

🧩

Framework Friendly

Works with plain HTML, CDN, Laravel, React, Vue, Vite, and Next.js client components.

Demo Experience

Add tools without rebuilding your table.

Add one configuration div, load CSS and JS, then call createHgmAnyTools().mount().

1

Install or use CDN

Use npm package or direct CDN from jsDelivr/unpkg.

2

Add labels config

Define default labels like Urgent, Hold, Recheck, and Done.

3

Use Ctrl / Alt + T

Open the mode chooser and start copying or labeling content.

Shortcut Guide
Active
Open Tools Menu Ctrl + T
Fallback Open Alt + T
Copy Selected Ctrl + C
Clear Labels Ctrl + Shift + X
Disable Mode Ctrl + X
Installation

Start with npm or CDN.

NPM

Recommended 
npm install hagematech-copas-labels
import createHgmAnyTools from "hagematech-copas-labels";
import "hagematech-copas-labels/style.css";

createHgmAnyTools().mount();

CDN Latest

Fast Demo 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/hagematech-copas-labels@latest/src/style.css">

<script type="module">
    import createHgmAnyTools from "https://cdn.jsdelivr.net/npm/hagematech-copas-labels@latest/src/index.js";

    createHgmAnyTools().mount();
</script>
Quick Usage

Drop it into any page.

index.html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/hagematech-copas-labels@latest/src/style.css">

<div
    data-hgm-any-tools
    data-page-key="auto"
    data-default-color="#2563eb"
    data-labels='[
        {"name":"Urgent","color":"#dc2626"},
        {"name":"Hold","color":"#f59e0b"},
        {"name":"Recheck","color":"#2563eb"},
        {"name":"Done","color":"#16a34a"}
    ]'>
</div>

<div class="hgm-table-tools-shortcut-badge">
    Press <kbd>Ctrl</kbd> / <kbd>Alt</kbd> + <kbd>T</kbd>
</div>

<table>
    <thead>
        <tr>
            <th>Deal ID</th>
            <th>Customer</th>
            <th>Status</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <td data-hgm-label-key="deal-1001-id">CRM-1001</td>
            <td data-hgm-label-key="deal-1001-customer">Nusantara Capital Bank</td>
            <td data-hgm-label-key="deal-1001-status">Completed</td>
        </tr>
    </tbody>
</table>

<script type="module">
    import createHgmAnyTools from "https://cdn.jsdelivr.net/npm/hagematech-copas-labels@latest/src/index.js";

    createHgmAnyTools().mount();
</script>
